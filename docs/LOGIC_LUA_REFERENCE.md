## Item Definition Fields

| Field                  | Type        | Description                                                                                                                                                                                                                                                                    |
|------------------------|-------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `name`                 | string      | **Required.** Control name, referenced in assignments                                                                                                                                                                                                                          |
| `controlID`            | number      | Numeric ID for use in CS callbacks                                                                                                                                                                                                                                             |
| `label`                | string      | Display label                                                                                                                                                                                                                                                                  |
| `objectType`           | string      | `'Button'`, `'Knob'`, `'VFader'`, `'Wheel'`, `'Keyboard'`, `'Drumpad'`, `'Sustain Pedal'`                                                                                                                                                                                      |
| `midiType`             | string      | `'Momentary'`, `'Absolute'`, `'Alternating'`, `'Relative2C'`, `'RelativeSM'`, `'Single'`, `'Keyboard'`, `'Ignore'`                                                                                                                                                                             |
| `midi`                 | table       | MIDI bytes. Can be 3 bytes `{0x90, 0x3A, MIDI_LSB}` or 6 bytes for multi-message                                                                                                                                                                                               |
| `midiTouched`          | table       | MIDI message for touch detection                                                                                                                                                                                                                                               |
| `inport`               | string      | MIDI input port name                                                                                                                                                                                                                                                           |
| `outport`              | string      | MIDI output port name                                                                                                                                                                                                                                                          |
| `hasFeedback`          | bool        | Whether Logic sends numeric feedback to this control                                                                                                                                                                                                                           |
| `hasFeedbackValueText` | bool        | Whether Logic sends text feedback to this control                                                                                                                                                                                                                              |
| `fbType`               | constant    | Feedback type: `FB_OFF`, `FB_HOR_SINGLE`, `FB_HOR_LTOR_BAR`, `FB_HOR_RTOL_BAR`, `FB_HOR_CNTR_BAR`, `FB_HOR_Q`, `FB_VERT_ASC`, `FB_VERT_DESC`, `FB_TEXTONLY`, `FB_AUTO`, `FB_ONIFDEF`, `FB_OFFIFDEF`, `FB_ALWAYSON`, `FB_ALWAYS2`, `FB_MULTILED`, `FB_TRIM`, `FB_MIN`, `FB_MAX` |
| `valueMode`            | constant    | `kAssignDirect`, `kAssignScaled`, `kAssignRelative`, `kAssignRotate`, `kAssignToggle`, `kAssignFixed`, `kAssignXOR`, `kAssignGroup1`, `kAssignCurGroup`, `kAssignUnsigned`, `kAssign1sComplement`, `kAssign2sComplement`, `kAssignSignMagnitude`                               |
| `maxVal`               | number      | Maximum value                                                                                                                                                                                                                                                                  |
| `minVal`               | number      | Minimum value                                                                                                                                                                                                                                                                  |
| `multiply`             | number      | Value multiplier                                                                                                                                                                                                                                                               |
| `keyCmd`               | number      | Key command ID (can be in item OR assignment)                                                                                                                                                                                                                                  |
| `trackParam`           | constant    | Track parameter (can be in item OR assignment)                                                                                                                                                                                                                                 |
| `CSTrack`              | bool/number | `true` or `0` = selected track                                                                                                                                                                                                                                                 |
| `startKey`             | number      | First MIDI note (Keyboard type)                                                                                                                                                                                                                                                |
| `numberKeys`           | number      | Number of keys (Keyboard type)                                                                                                                                                                                                                                                 |
| `keyRepeat`            | bool        | Whether to repeat on hold                                                                                                                                                                                                                                                      |
